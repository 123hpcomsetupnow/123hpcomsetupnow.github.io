Complete HP Printer Setup using 123.hp.com/setup


  1. Open 123.hp.com/setup in your web browser.
  2. Enter the model number of your HP printer in the box provided and click "Submit".
  3. The available software and drivers for your printer model will appear. Download the required driver by clicking the "Download" button.
  4. Open the downloaded file and install the driver and software on your computer by following the on-screen instructions.
  5. During the installation, you will be asked to connect the printer to the computer via Wi-Fi or USB. Set up the connection by following the instructions.
  6. Confirm all settings and connections, and you will see a success message when the setup is complete.
  7. Now your HP printer is ready and you can use it to print, scan and copy.

Troubleshooting Common HP Printer Problems


1. Printer connectivity problem:

  • Make sure your printer is properly connected to the Wi-Fi network or the cable is connected correctly.
  • Restart the printer and router and try connecting again.

2. Print quality is poor:

  • Install the ink or toner cartridge correctly and refill or replace it.
  • Use the head cleaning tool in the printer settings.

3. Printer is slow:

  • Try printing in draft mode.
  • Make sure the correct paper size and type is selected for printing.

4. Printer paper jam:

  • Gently remove the paper stuck inside the printer.
  • Load paper correctly in the printer tray and do not put too much paper.

5. Driver problem:

  • Download and install the latest driver for the printer from 123.hp.com/setup.
  • Uninstall existing driver and install new driver.

6. Printer showing offline:

  • Go to the printer settings on the computer and set it to "Set as default printer" and uncheck the "Use printer offline" option.
  • Confirm that your printer and computer are connected to the same network.

7. Not printing:

  • There may be any pending or stuck print jobs in the print queue, clear it.
  • Update the printer's firmware.

8. Scanner not working:

  • Update the scanner's software.
  • Check that scanner is connected to the computer correctly.
With these measures, you can easily troubleshoot common HP printer problems.
